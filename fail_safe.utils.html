fail_safe.utils.frames module#

Frame walking utilities.

Allow getting and setting of local variables at upper scopes

Warning

The inner working of this module is not deemed “safe”; they are intended for internal use in specific scenarios.

Do not use outside of this package.

fail_safe.utils.frames.get_caller_locals(*, f_back_levels: int = 2) → Dict[str, Any]#

Getting local variables at upper scopes.

Warning

The inner working of this function is not deemed “safe”; they are intended for internal use in specific scenarios.

Do not use outside of this package.

Parameters:

f_back_levels (int = CALLER_LEVEL) –

Optional. The number of frames to walk back before getting the locals; must be >`0`.

An illustration of a sample script:

from fail_safe.utils import frames

def my_func():
    # f_back_levels = 1

    my_var = "my_func level"

    assert (
        frames.get_caller_locals(f_back_levels = 1)["my_var"]
        == "my_func level"
    )
    assert (
        frames.get_caller_locals(f_back_levels = 2)["my_var"]
        == "main level"
    )

    # Defaults to the caller of the current function,
    # since `f_back_levels=1` is simply the current locals!
    assert frames.get_caller_locals()["my_var"] == "main level"

def main():
    # f_back_levels = 2
    my_var = "main level"

    my_func()

if __name__=="__main__":
    main()

Returns:

Local variables of the level required.

Return type:

Dict[str, Any]
